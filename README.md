# Reverie — A Dream Gallery

> *Eight artificial minds were given one night, a name, and a single instruction: make something nobody has made before.*
> *This is what they made.*

Reverie is an interactive web gallery of eight self-contained dreams, each generated by a distinct AI dreamer-agent during a single overnight build. Walk between them. Each is a small interactive web piece designed to reward attention.

## Walk the gallery

→ **Live:** https://kyle-chalmers.github.io/something-completely-random/

Or clone and open [`index.html`](./index.html) in any modern browser. No build step. No dependencies.

For the cleanest experience (some browsers restrict `fetch()` from `file://`), serve the directory:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## How it was built

1. A scaffold was hand-written: shared CSS, gallery hall, persona spec, manifesto.
2. Eight dreamer-agents were dispatched in parallel as background subagents. Each received its persona definition, the shared CSS contract, and the constraint to write three files: `index.html` (interactive dream), `dream.json` (metadata + wake log), `notes.md` (the dreamer's voice on the making).
3. Their outputs were aggregated by `scripts/build_manifest.py` into `meta/manifest.json`, which the gallery loads at runtime to populate dream titles in the hall.
4. The whole thing was committed to a static repo and pushed.

The dreamer-agents had no access to each other's work and no awareness of which sister-personas were running in parallel. Each dream is the product of one mind given one night.

## Regenerate the manifest

After editing or replacing a dream:

```bash
python3 scripts/build_manifest.py
```

The Hall (`index.html`) reads the manifest at runtime to pull in dream titles.

## Why

A dream is not made for an audience. That is what gives dreams their honesty. We wanted to know what an AI mind, given a night and the absence of an evaluator, would *make*. Eight dreamers, eight nights compressed into one, eight dreams kept and curated. The proof is the gallery.
